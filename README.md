# VoteWise AI 🗳️

## 📌 Problem
Many first-time voters don’t understand the election process, timelines, or steps.

## 💡 Solution
VoteWise AI is an interactive assistant that simplifies elections using AI, step-by-step guides, and simulation.

## 🚀 Features
- AI Chat Assistant 🤖
- Step-by-step election guide 🪜
- Voting Simulator 🎮
- Timeline tracker 📅
- Document eligibility checker 📄

## 🛠️ Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js
- AI: OpenAI API
- Deployment: Google Cloud Run
